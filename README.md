# Palhanos

Palhanos é um blog e portfólio pessoal criado para reunir anotações, estudos e projetos em um único lugar.

A ideia surgiu da necessidade de ter um ambiente simples para organizar conteúdos de diferentes áreas e torná-los acessíveis de qualquer lugar. Todo o conteúdo é escrito em Markdown e convertido dinamicamente para páginas da aplicação.

O projeto foi desenvolvido com foco em simplicidade, desempenho e facilidade de manutenção.

## Tecnologias

- Next.js
- React
- Tailwind CSS
- JavaScript
- Markdown

## Funcionalidades

- Publicação de artigos em Markdown
- Geração dinâmica de páginas por slug
- Sistema de tópicos e categorização
- Navegação entre artigos
- Layout responsivo
- Deploy automatizado com Vercel

## Estrutura do conteúdo

Os artigos são armazenados como arquivos Markdown, permitindo que novos conteúdos sejam publicados sem alterações na interface da aplicação.

## Deploy

O projeto está disponível em:

https://palhanos.vercel.app

## Objetivo

Mais do que um portfólio, o Palhanos funciona como um espaço para registrar estudos, documentar projetos e compartilhar conhecimento adquirido ao longo da graduação e de projetos pessoais.
