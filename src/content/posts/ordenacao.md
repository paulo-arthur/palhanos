# Algoritmos de Ordenação

Algoritmos de ordenação são basicamente um tipo de processo que organiza um array de númeors, caracteres ou strings em uma ordem crescente ou alfabética. Sua utilidade revela-se gigante quando utilizamos a organização de um conjunto para simplificar processos, como na busca binária.

## Tipos de Algoritmos de Ordenação
Existem vários tipos de algoritmos de ordenação que variam em eficácia, espaço de memória necessário e método de funcionamento. Estes são alguns exemplos:
- Selection Sort
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort

Alguns algoritmos podem ordenar uma lista sem criar outra inteiramente nova. Chamamos um algoritmo assim de algoritmo de ordenação  *in loco* e ele requer um espaço adicional O(1) constante para a ordenação. Quando os algoritmos não são *in loco*,  uma outra lista é criada durante a ordenação.
